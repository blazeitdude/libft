## Compile
to build a project<br/>
`$ make`<br/>
to build the bonus part<br/>
`$ make bonus`<br/>
to delete %.o files<br/>
`$ make clean`<br/>
to delete %.o and the library<br/>
`$ make fclean`<br/>
to rebuild the project<br/>
`$ make re`<br/>
